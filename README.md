Keyboard Shortcuts

| Action         | Windows/Linux Default | macOS Default       |
|----------------|-----------------------|---------------------|
| Play / Pause   | Ctrl + Shift + P      | Command + Shift + P |
| Next Track     | Ctrl + Shift + N      | Command + Shift + N |
| Previous Track | Ctrl + Shift + B      | Command + Shift + B |


To change shortcuts:

Visit brave://extensions/shortcuts
